# Tarusa Main Website

A modern, interactive website for Tarusa built with React and Vite, featuring 3D model integration and smooth animations.

## 🚀 Features

- **Modern UI/UX**
  - Clean, minimalist design
  - Smooth animations and transitions
  - Custom Gilroy font integration
  - Responsive layout (in progress)

- **Interactive Elements**
  - Text scrambling animations
  - 3D model viewer (coming soon)
  - Custom hover effects
  - Smooth scrolling

- **Performance Optimized**
  - Built with Vite for fast development and builds
  - Optimized asset loading
  - Efficient animations using CSS transitions

## 🎨 Brand Colors

## 🛠️ Tech Stack

- React 18
- Vite
- Tailwind CSS
- Three.js (coming soon)
- Framer Motion (coming soon)

## 📦 Installation

1. Clone the repository:
```bash
git clone [repository-url]
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
